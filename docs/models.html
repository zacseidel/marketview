Common Mechanics
Universe

All models screen against the S&P 500 and/or S&P 400, sourced from weekly Wikipedia scrapes (~901 active tickers total). Composition is reconciled every Sunday; additions and removals update the universe file within 24 hours.

Evaluation cycle

The pipeline runs on Tuesday and Friday at 11:00 AM MT via GitHub Actions. Each run re-scores every eligible ticker and produces a new set of holdings.

Exit rules

Two exit mechanisms are in use across models:

Time-based (default)
A ticker is held for up to 10 trading days after the most recent run that recommended it. If re-picked, the clock resets. Once 10 days elapse without a re-pick, the position is marked sell.
Immediate price exit
Models that have a hard price condition (e.g. EMA violation) emit an explicit sell on the same run the condition is breached — no countdown. Currently used by Munger (EMA 15) and Repurchase (EMA 21).
Conviction score

Each model assigns a conviction between 0.0 and 1.0 using its own scheme (documented per model below). Conviction is used for display ranking only — it does not affect position sizing.

Return metric

All return calculations use log returns: log(price_t / price_t0). Log returns are additive over time and symmetric around zero, making them more suitable for model training and comparison than percentage returns.

Momentum Live
Strategy

Buys the top 5 S&P 500 stocks by trailing 12-month log return, with a rank-stability filter to avoid whipsawing in and out of high-volatility names.

Universe

S&P 500 only. Tickers missing a full 252-day price history are excluded.

Signal
Return window
252 trading days (≈ 12 months)
Metric
log(close_today / close_252d_ago)
Ranking
All valid S&P 500 tickers ranked descending by 12-month log return
Rank-stability filter

A ticker only qualifies if its current rank is equal to or better than its rank from the previous run. This removes names that are momentum leaders but losing ground — the first sign of a trend reversal. Tickers with no prior rank (first run, or newly added) pass automatically.

Selection

Top 5 tickers from the rank-stable set.

Conviction formula
conviction = 1 − (rank − 1) / (total_ranked − 1)

Rank 1 scores 1.0; the last-ranked ticker scores 0.0. Only top-5 tickers are held, so minimum conviction in the portfolio is approximately 0.99 (rank 5 of ~480 valid tickers).

Munger — Large-Cap Mean Reversion Live
Strategy

Inspired by Charlie Munger's principle of buying quality businesses at fair prices. Screens the 100 largest S&P 500 companies for recent dips to or below their 200-day SMA, then buys only if the price has recovered back above the 15-day EMA — confirming the dip was bought, not the start of a downtrend.

Universe

Top 100 S&P 500 tickers by market cap (from most recent constituent data). The full ranked list is saved each run as munger_universe.json; the Universe Changes tile on each report shows week-over-week entry/exit from this set.

Buy / hold signal

Both conditions must be true simultaneously:

SMA(200) dip
Price touched at or below its 200-day simple moving average at least once in the last 21 trading days
EMA(15) recovery
Current close is strictly above the 15-day exponential moving average
Exit logic

Two distinct exit paths, depending on why the model stops re-picking a holding:

EMA(15) broken
Immediate sell — emitted on the same run the violation is detected. Does not wait for the time-based countdown.
SMA(200) touch expired
Time-based exit — the dip opportunity has passed but price is still healthy. Held for up to 10 more trading days without a re-pick, then sold.
Conviction formula
pct_above_ema15 = (close − EMA15) / EMA15 conviction = min(pct_above_ema15 / 0.05, 1.0)

A stock 5% or more above its EMA(15) scores 1.0. The cap reflects that larger gaps often indicate overbought conditions rather than higher quality.

Repurchase — Buyback Yield Live
Strategy

Identifies companies aggressively buying back their own stock. Heavy, consistent buybacks signal management confidence in intrinsic value and reduce share count, mechanically increasing per-share earnings.

Universe

S&P 400 (S&P 500 expansion pending fundamentals backfill). Requires at least 4 quarters of shares-outstanding data.

Signal
Buyback rate
(shares_4q_ago − shares_now) / shares_4q_ago — trailing 12-month net reduction in share count
EMA filter
Current close must be above the 21-day EMA (avoid catching falling knives)
Selection
Top 5 tickers by buyback rate, EMA filter applied first
Exit logic

Two distinct exit paths, depending on why the model stops re-picking a holding:

EMA(21) broken
Immediate sell — emitted on the same run the violation is detected. Does not wait for the time-based countdown.
Dropped from top-5 ranking
Time-based exit — buyback yield is no longer top-ranked but price is still above EMA. Held for up to 10 more trading days without a re-pick, then sold.
Conviction formula

Normalized rank within the qualifying set: rank 1 scores 1.0, rank 5 scores approximately 0.0.

Watchlist Live
Strategy

A manually curated list of tickers with user-assigned conviction scores and notes. Useful for tracking thesis-driven positions that don't fit a purely quantitative screen.

Source

Reads from config/watchlist.yaml. Each entry specifies a ticker, conviction (0–1), and optional rationale notes. The model produces a holding record for each active ticker in the file.

Quant GBM v7 — XGBoost Factor Model Live
Strategy

A gradient-boosted tree model trained to predict 5-day forward raw returns. Scores all S&P 500 + S&P 400 tickers on Friday and picks the top 5 by predicted score.

Model details
Algorithm
XGBoost with enable_categorical=True
Target
5-day forward raw log return
Training universe
S&P 500 + S&P 400, 10 years of daily price history
Training filter
Thursdays only — aligns features with the Friday evaluation cycle
Train / val split
Walk-forward: most recent 2 years = validation, rest = training
Validation ICIR
2.983 (Information Coefficient × √252 / IC std dev)
Run cycle
Friday only — returns no picks on Tuesday runs
Feature count
46 numeric + 1 categorical (sector) = 47 total
Conviction formula
conviction = (score − min_score) / (max_score − min_score)

Scores are normalized within the top-5 selection. Rank 1 scores 1.0; rank 5 scores 0.0.

Anti-leakage rules
earn_ret_5d_to_20d
Masked to NaN within 20 trading days of the earnings event (the day-20 price is still in the future)
ni_qoq_growth
Clipped to ±3 to prevent extreme one-off quarters from dominating splits
ni_acceleration
Clipped to ±3 for the same reason
